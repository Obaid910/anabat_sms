# Anabat School Management System

A comprehensive multi-branch school management system for managing academic, administrative, financial, and communication activities.

## Project Overview

This system provides a centralized platform for school management with multi-branch support, featuring modules for:
- Admissions (Lead Management)
- Student Management
- Attendance Tracking
- Communication System
- Exams and Grading
- Billing and Fees
- HR and Payroll
- Accounting
- Inventory Management
- Reporting and Analytics

## Technology Stack

- **Backend**: Laravel (PHP)
- **Frontend**: React with Material-UI
- **Database**: PostgreSQL
- **API**: RESTful Architecture
- **Authentication**: JWT with role-based access

## Development Phases

1. Core System (Authentication, Multi-branch, Admissions)
2. Academic Management (Students, Attendance, Communication, Exams)
3. Financial Management (Billing, Payments, Accounting)
4. Staff & Resources (HR, Payroll, Inventory)
5. Reporting & Optimization

See the `docs` directory for detailed development guides.
