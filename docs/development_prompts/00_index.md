# Anabat School Management System - Development Prompts

This directory contains structured prompts for each phase of development of the Anabat School Management System. These prompts can be used with AI assistants to help implement specific features or modules of the system.

## How to Use These Prompts

1. Copy the prompt for the specific feature you want to implement
2. Paste it into your conversation with an AI assistant
3. The AI will help guide you through the implementation steps
4. Adapt the prompts as needed based on your specific requirements

## Customizing Prompts

Feel free to modify these prompts based on your specific requirements. You can:
- Add or remove steps
- Change the technology stack
- Adjust the implementation details
- Combine prompts for more comprehensive guidance

## Best Practices

- Break down large implementations into smaller tasks
- Use the prompts as a guide, not as strict instructions
- Always review and test the code provided by AI assistants
- Adapt the solutions to match your specific business requirements
- Document any deviations from the standard implementation
